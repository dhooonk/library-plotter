## 개요

TFT(박막 트랜지스터) 시뮬레이션 결과 파일(`.xlsx`, `.xls`, `.csv`)을 읽어  
**Transfer Curve(Vgs-Id)** 와 **Output Curve(Vds-Id)** 를 자동으로 플로팅하고,  
레퍼런스 데이터와 R² 비교 분석 후 결과를 엑셀 보고서로 내보냅니다.

## 개발배경

SmartSpice 시뮬레이션 결과를 매번 수작업으로 정리하고 그래프를 그리는 데 많은 시간이 소요되었습니다.  
파일 포맷이 일정하지 않아 헤더 위치나 데이터 시작 행이 달라지는 문제도 반복적으로 발생하였고,  
시뮬레이션 결과와 측정 데이터의 일치도를 정량적으로 비교할 수 있는 전용 도구가 필요했습니다.

## 목적

- 반복적인 TFT 특성 곡선 시각화 작업을 자동화
- 시뮬레이션 데이터와 측정 레퍼런스 간 R² 기반 정량 비교 지원
- 이상치 제거, 축 범위 조정 등 그래프 품질 관리 기능 제공
- 결과를 엑셀 보고서로 즉시 내보내는 워크플로 구축

## 구현사항

- 파일 내 헤더/데이터 시작 행 자동 감지 (SmartSpice `DataValue` 포맷 포함)
- Vd / Vg 파라미터별 곡선 자동 그룹화 및 오름차순 정렬
- 데이터 포인트 수 부족 그룹 자동 제외 (노이즈 필터링)
- 레퍼런스 곡선 선형 보간 후 R² 점수 산출 및 범례 표시
- 마우스 호버 시 x, y 좌표 실시간 어노테이션 표시
- 차트 우클릭으로 이상치 데이터 포인트 즉시 제거
- 분석 실행 후 실제 데이터 범위로 축 Min/Max 자동 입력
- 이상치 삭제 패널에 클릭 기반 다중 선택 테이블 제공
- Raw Data + 차트 이미지를 포함한 엑셀 보고서 자동 생성

## 사용설명서

### 1. 기본 분석 흐름

| 단계 | 설명 |
|------|------|
| ① 탭 선택 | 상단에서 **Transfer Curve** 또는 **Output Curve** 탭 클릭 |
| ② 파일 선택 | `메인 데이터 선택` — 분석할 `.xlsx` / `.csv` 파일 선택 |
| ③ (선택) Ref 선택 | `비교 데이터(Ref) 선택` — R² 비교용 레퍼런스 파일 선택 |
| ④ 분석 실행 | `📊 분석 실행` — 데이터 파싱 및 차트 자동 생성 |
| ⑤ 결과 저장 | `📥 엑셀로 저장` — Raw Data + 차트 포함 보고서 저장 |

### 2. 축 범위 조정

- `분석 실행` 후 X/Y 축 **Min / Max** 필드에 실제 데이터 범위가 자동 입력됩니다.
- 원하는 범위로 수정 후 `🔄 차트 새로 그리기` 를 눌러 적용합니다.
- **Y축 로그 스케일** 체크박스로 Log / Linear 전환이 가능합니다.

### 3. 이상치 삭제

이상치 삭제 패널에서 마우스로 간편하게 데이터를 제거할 수 있습니다.

#### 방법 A — 테이블에서 선택 삭제 (권장)
1. **파라미터 값** 드롭다운에서 대상 곡선(Vd 또는 Vg)을 선택합니다.
2. 데이터 목록 테이블에서 제거할 행을 클릭합니다.
   - 단일 클릭: 한 행 선택
   - `Ctrl + 클릭`: 여러 행 동시 선택
   - `Shift + 클릭`: 범위 선택
3. `✂️ 선택 삭제` 버튼을 클릭합니다.
4. 차트가 즉시 갱신되고, 제외된 행은 테이블에 **"제외"** 로 표시됩니다.

#### 방법 B — 차트에서 우클릭 삭제
- 차트 위의 삭제할 데이터 포인트 위에서 **우클릭**하면 가장 가까운 포인트가 자동으로 제거됩니다.

#### 전체 복원
- `↩ 전체 복원` 버튼으로 모든 제외 항목을 원래대로 되돌립니다.

### 4. 엑셀 저장

- **저장 파일명**: 기본값은 `{입력파일명}_result.xlsx`이며 자유롭게 변경 가능합니다.
- **저장 경로 선택**: 원하는 폴더를 선택하거나, 기본값(입력 파일과 동일 경로)을 사용합니다.
- 저장 완료 후 파일을 바로 열지 선택할 수 있습니다.
- 결과 파일에는 **Raw Data 시트** (파라미터별 x·y 데이터)와 **Chart 시트** (고화질 차트 이미지)가 포함됩니다.

### 5. 로그 패널

- 왼쪽 하단 **로그** 카드에 파싱 결과, 이상치 제거 기록, 저장 경로 등이 순서대로 기록됩니다.
- 오류 발생 시에도 로그에 상세 내용이 표시되어 원인 파악에 활용할 수 있습니다.

## 주요 기능

- **자동 데이터 감지**: 헤더 위치와 무관하게 숫자 데이터 시작 행 자동 탐색
- **파라미터별 곡선 플로팅**: Vd / Vg 값 기준으로 곡선 자동 분리·정렬
- **레퍼런스 비교 (R²)**: 메인·레퍼런스 곡선 오버레이 및 일치도 점수 표시
- **호버 좌표 표시**: 마우스 위치의 x, y 값을 차트에 실시간 표시
- **클릭 기반 이상치 삭제**: 테이블에서 클릭(다중 선택)으로 직관적 이상치 제거
- **우클릭 이상치 제거**: 차트 위에서 우클릭으로 데이터 포인트 즉시 삭제
- **축 범위 자동 입력**: 분석 실행 시 실제 데이터 범위로 Min/Max 자동 채움
- **엑셀 보고서 내보내기**: Raw Data 시트 + 고화질 차트 이미지 시트 생성

## 기술 스택

- **GUI**: Python tkinter, matplotlib (TkAgg 백엔드)
- **데이터 처리**: pandas, numpy
- **분석**: scipy (선형 보간), scikit-learn (R²)
- **보고서**: openpyxl
