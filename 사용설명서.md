## 1. 프로그램 개요

**TFT Curve Analyzer**는 SmartSpice로 시뮬레이션한 TFT(박막 트랜지스터) 데이터를  
엑셀(.xlsx, .xls) 및 CSV(.csv) 파일로부터 읽어 두 가지 특성 곡선을 자동으로 시각화하고,  
다른 데이터 레퍼런스와 비교(R² 검증)하여 결과를 다시 파일로 저장하는 Python GUI 분석 도구입니다.

### 지원하는 분석 모드

| 모드 | 설명 | X축 | Y축 | 파라미터 |
|---|---|---|---|---|
| **Transfer Curve** | 게이트 전압에 따른 드레인 전류 특성 | Vgs (V) | \|Id\| (A) | Vd (드레인 전압) |
| **Output Curve** | 드레인 전압에 따른 드레인 전류 특성 | Vds (V) | Id (A) | Vg (게이트 전압) |

## 2. 설치 및 실행 방법

### 사전 요구사항

- Python 3.8 이상
- 아래 패키지가 설치되어 있어야 합니다:

| 패키지 | 버전 | 역할 |
|---|---|---|
| pandas | ≥ 1.5.0 | 데이터 파싱 등 (엑셀/CSV) |
| openpyxl | ≥ 3.0.0 | 엑셀 저장 |
| matplotlib | ≥ 3.5.0 | 그래프 플로팅 |
| scipy | 최선 | 선형 보간(Interpolation) |
| scikit-learn | 최선 | 의사 결정/R-squared 판별 지표 |

### 패키지 설치

```bash
# 프로젝트 폴더로 이동
cd tr-fitting-xls

# 가상환경 생성 (선택사항, 권장)
python -m venv venv
source venv/bin/activate   # macOS/Linux
venv\Scripts\activate      # Windows

# 필요 패키지 한 번에 설치
pip install -r requirements.txt
```

### 프로그램 실행

```bash
python main.py
```

실행 후 1400×800 크기의 GUI 창이 열립니다.

## 4. 입력 데이터 형식

### 4-1. 엑셀 파일 구조

프로그램은 **A, B, C, D 4개의 열**을 사용합니다.  
열의 의미는 Transfer Curve와 Output Curve에서 다릅니다.

#### Transfer Curve용 열 배치

| 열 | 내용 | 단위 | 비고 |
|---|---|---|---|
| A | (무시) | — | 임의의 값, 프로그램이 읽지 않음 |
| B | **Vg** (게이트 전압) | V | X축 데이터 |
| C | **Vd** (드레인 전압) | V | 그룹화 파라미터 |
| D | **Id** (드레인 전류) | A | Y축 데이터 |

#### Output Curve용 열 배치

| 열 | 내용 | 단위 | 비고 |
|---|---|---|---|
| A | (무시) | — | 임의의 값, 프로그램이 읽지 않음 |
| B | **Vd** (드레인 전압) | V | X축 데이터 |
| C | **Vg** (게이트 전압) | V | 그룹화 파라미터 |
| D | **Id** (드레인 전류) | A | Y축 데이터 |

### 4-2. 헤더 행 자동 감지

- 엑셀 파일 내 **헤더 행의 위치가 달라도 자동으로 감지**합니다.
- 알고리즘: B, C, D 열이 모두 숫자인 첫 번째 행을 데이터 시작점으로 판단합니다.
- 즉, 파일 상단에 제목, 설명 등 텍스트 행이 몇 줄 있어도 문제없습니다.

### 4-3. 예시 데이터 (Transfer Curve)

```
A         B(Vg)    C(Vd)    D(Id)
시뮬레이션  -5       0.1      -1.23E-12
결과       -4       0.1      -3.45E-12
           -3       0.1      -9.87E-11
           ...
           -5       1.0      -5.11E-11
```

### 4-4. 지원 파일 형식

| 형식 | 확장자 |
|---|---|
| Excel 2007 이상 | .xlsx |
| Excel 97-2003 | .xls |
| CSV (쉼표 구분) | .csv |

## 5. Transfer Curve 탭 사용법

Transfer Curve 탭에서는 **Vgs(게이트-소스 전압) 대비 드레인 전류(Id)** 특성을 분석합니다.  
각 Vd(드레인 전압) 값별로 별개의 곡선이 그려집니다.

### 분석 순서

#### Step 1 — 탭 선택

상단에서 **📈 Transfer Curve** 탭을 클릭합니다.

#### Step 2 — 엑셀 파일 선택

1. 좌측 패널의 **📂 입력 파일** 카드에서  
   `엑셀 파일 선택 (.xlsx / .xls)` 버튼을 클릭합니다.
2. 파일 탐색기에서 분석할 엑셀 파일을 선택합니다.
3. 선택 완료 시:
   - 파일명이 카드 상단에 **초록색**으로 표시됩니다.
   - 저장 파일명이 `원본파일명_result.xlsx`로 자동 설정됩니다.
   - 저장 경로가 입력 파일과 같은 폴더로 자동 설정됩니다.

#### Step 3 — 옵션 설정

**⚙️ 옵션** 카드의 `Y축 로그 스케일 (Log Scale)` 체크박스:

| 설정 | 권장 상황 |
|---|---|
| ✅ 체크 (기본값 ON) | Id 값이 매우 작거나 범위가 큰 경우 (서브스레시홀드 영역 포함) |
| ☐ 해제 | Id 범위가 좁고 선형 표시가 적합한 경우 |

> **팁:** Transfer Curve는 일반적으로 로그 스케일이 권장됩니다.  
> 서브스레시홀드 기울기(SS) 등 특성 파악에 유리합니다.

#### Step 4 — 분석 실행

1. **▶ 실행** 카드에서 `📊 분석 실행` 버튼을 클릭합니다.
2. 백그라운드 스레드에서 파싱이 수행되므로 UI가 멈추지 않습니다.
3. 분석 완료 시:
   - 우측 차트 영역에 Transfer Curve가 표시됩니다.
   - 로그 패널에 감지된 Vd 값 목록이 출력됩니다.
   - 차트 상단 바에 `Vd 값: Xv, Yv, ...` 형태로 파라미터가 표시됩니다.

#### Step 5 — (선택) 로그 스케일 변경

분석 완료 후에도 `Y축 로그 스케일` 체크박스를 토글하면  
저장된 데이터를 다시 파싱하지 않고 **차트만 즉시 재그리기**합니다.

## 6. Output Curve 탭 사용법

Output Curve 탭에서는 **Vds(드레인-소스 전압) 대비 드레인 전류(Id)** 특성을 분석합니다.  
각 Vg(게이트 전압) 값별로 별개의 곡선이 그려집니다.

### 분석 순서

#### Step 1 — 탭 선택

상단에서 **📉 Output Curve** 탭을 클릭합니다.

#### Step 2 — 엑셀 파일 선택

Transfer Curve와 동일한 방법으로 파일을 선택합니다.  
단, **열 배치가 다릅니다** (B=Vd, C=Vg, D=Id). 반드시 올바른 형식의 파일을 사용하세요.

#### Step 3 — 옵션 설정

Output Curve의 **기본값은 로그 스케일 OFF(선형)**입니다.  
Output Curve는 일반적으로 선형 스케일로 표시하는 것이 자연스럽습니다.

| 설정 | 권장 상황 |
|---|---|
| ☐ 해제 (기본값 OFF) | 일반적인 Id-Vd 출력 특성 표시 |
| ✅ 체크 | 낮은 전류 영역도 관찰하고 싶을 때 |

#### Step 4 — 분석 실행

Transfer Curve와 동일하게 `📊 분석 실행` 버튼을 클릭합니다.  
분석 완료 시 Vg 별 Vd-Id 곡선이 차트에 그려집니다.

## 7. 결과 저장 (엑셀 내보내기)

### 저장 경로 및 파일명 설정

**💾 저장 설정** 카드에서 설정합니다.

| 항목 | 설명 |
|---|---|
| **저장 파일명** | 저장할 파일 이름 (`.xlsx` 확장자 자동 추가) |
| **저장 경로 선택** | 버튼 클릭 시 폴더 선택 다이얼로그 표시 |
| **경로 표시** | 현재 선택된 저장 폴더 경로 표시 |

> 파일 미선택 시 기본 경로는 사용자 홈 디렉토리(`~/`)입니다.  
> 입력 파일 선택 시 자동으로 같은 폴더로 업데이트됩니다.

### 저장 실행

1. `분석 실행` 이후에 `📥 엑셀로 저장` 버튼을 클릭합니다.
2. 저장 완료 시 알림창이 표시되며, "지금 파일을 여시겠습니까?" 질문이 나타납니다.
   - **예**: 저장된 엑셀 파일을 즉시 엽니다.
   - **아니요**: 저장만 하고 닫습니다.

> ⚠️ **주의:** `분석 실행` 없이 `엑셀로 저장`을 누르면 경고 메시지가 표시됩니다.  
> 반드시 분석을 먼저 실행한 후 저장하세요.

## 9. 출력 엑셀 파일 구조

### Transfer Curve 출력 파일

#### 시트 1: `Raw Data`

- 각 Vd 값별로 **2개의 열** (Vgs, Id)이 나란히 배치됩니다.
- 행 1: **병합 헤더** — `Vd = X V` (진한 파랑 배경)
- 행 2: **서브 헤더** — `Vgs (V)`, `Id (A)` (미디엄 파랑)
- 행 3 이하: **데이터** — 짝수 행은 연한 파랑 배경으로 교대 표시
- 행 고정(Freeze): 3행부터 스크롤 (헤더 2행 고정)
- Vgs 값: 소수점 3자리 (`0.000`)
- Id 값: 과학적 표기 (`0.000E+00`)

예시:

```
|   Vd = 0.1 V    |   Vd = 1 V      |
| Vgs(V) | Id(A)  | Vgs(V) | Id(A)  |
| -5.000 | 1.23E-12 | ...  |        |
| -4.000 | 3.45E-12 | ...  |        |
```

#### 시트 2: `Transfer Curve Chart`

- 차트 이미지 (700×500 픽셀)가 `B4` 셀에 삽입됩니다.
- 격자선 없음 (깔끔한 표시)
- 제목 행: "TFT Transfer Curve (Vgs-Id)"
- 스케일 정보: "Y축: 로그 스케일 (Log) / 선형 스케일 (Linear)"

### Output Curve 출력 파일

#### 시트 1: `Raw Data`

- 각 Vg 값별로 **2개의 열** (Vds, Id)이 나란히 배치됩니다.
- 구조는 Transfer Curve와 동일합니다.

#### 시트 2: `Output Curve Chart`

- 차트 이미지가 삽입됩니다.
- 제목: "TFT Output Curve (Vd-Id)"

## 10. 비교 대상 데이터 분석 (Ref / R-squared)

TFT Curve Analyzer는 메인 데이터와 함께 **비교 기준이 되는 데이터(Ref)**의 일치율을 확인할 수 있습니다.

### 사용법
1. **📂 입력 파일** 카드에서 `비교 데이터(Ref) 선택` 버튼을 눌러 두 번째 데이터를 엽니다.
2. `📊 분석 실행`을 시작합니다.
3. 동일한 모델 매개변수 (Vd 혹은 Vg)가 매칭되면 차트에 **점선 형태**로 그려지며 오버레이 됩니다.
4. 범례 텍스트에 `[R²: 0.99x]` 와 같은 방식으로 일치율 수치가 산출되어 표시됩니다. (최대값 1.0)

## 11. 주의사항 및 FAQ

### 주의사항

> ⚠️ **열 순서를 반드시 맞춰 주세요.**  
> 프로그램은 엑셀의 B, C, D 열만 읽으며, 각 탭마다 열의 의미가 다릅니다.  
> Transfer용 파일을 Output 탭에서 열거나 그 반대를 수행하면 잘못된 결과가 나옵니다.

> ⚠️ **데이터에 빈 셀이 있으면 해당 행은 자동으로 제외됩니다.**

> ⚠️ **Vd/Vg 고유 값은 소수점 6자리 반올림 기준으로 구분합니다.**  
> 부동소수점 오차(예: 0.9999999 vs 1.0)도 자동으로 같은 값으로 처리됩니다.

> ⚠️ **이상치 필터링(Noise Filtering):**  
> Vg 또는 Vd 구간 중 데이터가 5개 미만인 구간(예: 쓰레기값 101V 등)은 렌더링되지 않습니다.

### FAQ

**Q. "데이터 시작 행을 찾을 수 없습니다" 오류가 납니다.**  
A. B, C, D 열에 숫자 데이터가 있는 행이 하나도 없는 경우입니다.  
   파일이 올바른 형식인지, 데이터가 B/C/D 열에 있는지 확인하세요.

**Q. 비교 데이터(R-squared)가 안 나타납니다.**  
A. 메인 파일의 Vd(또는 Vg) 값과 비교 파일의 Vd(또는 Vg) 값이 `1개도 교차`하지 않으면 R²를 연산할 수 없습니다.

**Q. 엑셀로 저장 후 파일이 열리지 않습니다.**  
A. Windows에서 `os.startfile()`, macOS에서 `open` 명령어로 파일을 엽니다.  
   파일이 다른 응용프로그램에서 열려 있으면 저장이 실패할 수 있습니다.

### 데이터 처리 알고리즘

1. **헤더 감지**: B, C, D 열이 모두 숫자인 첫 번째 행 탐색 (O(n), 행 순차 탐색)
2. **Vd/Vg 고유 값 추출**: 소수 6자리 반올림 + 집합(set)으로 중복 제거
3. **그룹화**: 각 고유 파라미터 값에 대해 절댓 허용오차 1e-9 비교 마스킹
4. **정렬**: X축 데이터(Vg 또는 Vd)는 오름차순 정렬 후 저장
